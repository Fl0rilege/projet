# Projet de E3
projet tatouage du son - système de transmission audio

L’objectif de ce projet est de créer un système de communication permettant de transmettre de l’information, en l’occurrence une image et un texte via la diffusion d’une musique. Afin de parvenir à cet objectif, nous allons utiliser la technique du tatouage du son pour dissimuler les données de notre image dans les hautes fréquences, qui sont inaudibles à l’oreille humaine. Ensuite, à l’aide d’un micro les capter, et après analyse retrouver l’image transmise. Pour ce faire nous disposons de deux cartes Raspberry pi, un micro et des enceintes.
